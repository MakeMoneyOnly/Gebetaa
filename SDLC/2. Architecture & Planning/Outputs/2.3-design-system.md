## 2. Layout Foundations

### App Shell
*   **Max Width:** `512px` (`max-w-lg`) to maintain a consistent mobile-optimized feel on all screens.
*   **Centering:** `mx-auto` for the main content area.
*   **Feed Layout:** Two-column masonry with a `-mt-8` offset on the first column.

### Interaction Utilities
*   **Shadows:** 
    *   `shadow-float`: `0 10px 30px rgba(0,0,0,0.15)`
    *   `shadow-float-strong`: `0 20px 40px rgba(0,0,0,0.25)`

---

## 3. Responsive Strategy
*   **Mobile-First:** All visual components designed for 1:1 mobile parity.
*   **Viewport:** `-webkit-tap-highlight-color: transparent` for native app feel.
